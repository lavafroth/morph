# Pseudo Mesh Morphing

A sleight of hand that makes it look like one mesh smoothly morphs into another.

## Getting Started

```sh
npm ci --include=dev
npx vite
```

You may change the meshes to anything you wish, including detailed meshes like teapots.
The default is a glock morphing into an icosahedron.
